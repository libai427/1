#简单计算器 - Tkinter 实现
这是一个使用 Python 和 Tkinter 库实现的简单计算器应用程序。
 功能特点
- 基本算术运算：加(+)、减(-)、乘(×)、除(÷)
- 支持小数运算
- 清除功能(AC)
- 错误处理（除零错误、无效输入）
- 响应式布局，按钮可自适应窗口大小
 运行要求
- Python 3.x
- Tkinter 库（通常随Python标准库安装）
 使用方法
1. 使用数字按钮输入数字
2. 使用运算符按钮进行运算
3. 按 "=" 计算结果显示
4. 按 "AC" 清除所有输入
 代码结构
- 主程序使用 Tkinter 创建 GUI 界面
- `button_click()` 函数处理所有按钮点击事件
- 使用网格布局管理器进行界面布局
- 自动处理运算符替换（显示×÷但内部使用*/）
 自定义选项
你可以轻松修改以下内容：
- 修改 `root.geometry()` 调整窗口大小
- 修改 `font` 参数改变字体和大小
- 调整 `grid()` 参数改变按钮布局
 已知限制
- 不支持连续运算（每次计算后需要重新输入）
- 不支持括号运算
- 不支持退格功能
